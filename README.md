# 资金费率套利机器人 — 稳健版

> 本金 $10,000 | 1x 杠杆 | BTC + ETH | 30% 手续费返佣

---

## 收益测算（含返佣）

### 核心假设

```
本金:           $10,000
实际套利资金:    $8,500（预留 $1,500 应急/链上底仓）
仓位分配:       BTC $4,250 + ETH $4,250
杠杆:           1x（不加杠杆）
返佣比例:       30%
```

### 手续费明细

```
单次开仓手续费（以 $4,250 单仓为例）:
  现货 Taker:    $4,250 × 0.10%  =  $4.25
  合约 Taker:    $4,250 × 0.04%  =  $1.70
  单次开仓合计:                      $5.95

单次平仓手续费:                      $5.95

一轮开+平合计:   $5.95 × 2         = $11.90
返佣 (30%):     $11.90 × 30%      = $3.57
净手续费:       $11.90 - $3.57     = $8.33
```

### 场景一：保守（熊市 / 横盘）

```
平均费率:       0.01% / 8h
套利资金:       $8,500

每日费率收入:   $8,500 × 0.01% × 3 = $2.55
每月费率收入:   $2.55 × 30          = $76.50

换仓频率:       每月 2 次
每月净手续费:   $8.33 × 2 × 2仓    = $33.32
每月返佣:       $33.32 ÷ 0.7 × 0.3 = $14.28

链上底仓收益:   $1,500 × 8% ÷ 12   = $10.00

每月净收入:     $76.50 - $33.32 + $14.28 + $10.00 = $67.46
┌─────────────────────────────────┐
│   年化收益:  $67.46 × 12 = $809 │
│   年化利率:  ≈ 8.1%              │
└─────────────────────────────────┘
```

### 场景二：中性（常态市场）⭐ 最可能

```
平均费率:       0.02% / 8h
套利资金:       $8,500

每日费率收入:   $8,500 × 0.02% × 3 = $5.10
每月费率收入:   $5.10 × 30          = $153.00

换仓频率:       每月 3 次
每月净手续费:   $8.33 × 3 × 2仓    = $49.98
每月返佣:                           = $21.42

链上底仓收益:   $1,500 × 10% ÷ 12  = $12.50

每月净收入:     $153.00 - $49.98 + $21.42 + $12.50 = $136.94
┌──────────────────────────────────────┐
│   年化收益:  $136.94 × 12 = $1,643   │
│   年化利率:  ≈ 16.4%                  │
└──────────────────────────────────────┘
```

### 场景三：乐观（牛市情绪）

```
平均费率:       0.035% / 8h
套利资金:       $8,500

每日费率收入:   $8,500 × 0.035% × 3 = $8.93
每月费率收入:   $8.93 × 30           = $267.75

换仓频率:       每月 4 次
每月净手续费:   $8.33 × 4 × 2仓     = $66.64
每月返佣:                            = $28.56

链上底仓收益:   $1,500 × 12% ÷ 12   = $15.00

每月净收入:     $267.75 - $66.64 + $28.56 + $15.00 = $244.67
┌──────────────────────────────────────┐
│   年化收益:  $244.67 × 12 = $2,936   │
│   年化利率:  ≈ 29.4%                  │
└──────────────────────────────────────┘
```

### 返佣的影响

```
以中性场景为例，有无返佣的对比:

                    无返佣      有返佣(30%)    差额
年手续费支出:        $599.76    $599.76       —
年返佣收入:          $0         $179.93       +$179.93
年净手续费:          $599.76    $419.83       节省 $180
年化利率:            14.6%      16.4%         +1.8%
```

> 30% 返佣大约提升 1.5-2% 的年化收益。资金越大、换仓越频繁，返佣价值越高。

## 项目结构

```
funding-arb-bot/
├── config.yaml       # 配置文件（API 密钥、策略参数）
├── main.py           # 主控程序 + 调度器
├── monitor.py        # 费率监控（ccxt 只读，不影响返佣）
├── executor.py       # 下单引擎（Binance 官方 SDK，保护返佣）
├── position.py       # 仓位管理 + SQLite 存储
├── notifier.py       # Telegram 通知
├── requirements.txt  # Python 依赖
├── arbitrage.db      # 交易数据库（自动生成）
└── logs/             # 运行日志（自动生成）
```

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 编辑配置
cp config.yaml config.yaml.bak
vim config.yaml   # 填入 API Key、Telegram Token

# 3. 创建 Telegram Bot
#    - 找 @BotFather，发送 /newbot
#    - 记录 bot_token
#    - 找 @userinfobot 获取你的 chat_id

# 4. 启动（建议用 screen/tmux）
screen -S arb
python main.py

# 5. 后台运行后按 Ctrl+A+D 退出 screen
```

## 安全清单

```
[ ] API Key 只开启 现货交易 + 合约交易 权限
[ ] 关闭 提币权限（防止 Key 泄露资金被转走）
[ ] 设置 IP 白名单（只允许你的服务器 IP）
[ ] config.yaml 加入 .gitignore，不要提交到 Git
[ ] 服务器开启防火墙，只开放 SSH 端口
[ ] 定期检查 Telegram 通知是否正常
```
